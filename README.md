# Typegraph Core

[Metatype](https://github.com/metatypedev/metatype) standard library.

## Getting started

```
pip install typegraph-std
```

### Development

```shell
pre-commit install
python3 -m venv .venv
source .venv/bin/activate
poetry install
# sync with metatype
# pip install -e /path/to/metatype/typegraph/python
```
